# BlazorApp

## Pre-requisites

Install .NET WebAssembly build tools

```bash
dotnet workload install wasm-tools
```

## Create New

Create new Blazor App:

```bash
dotnet new blazorwasm -o BlazorApp --no-https -f net6.0
```
